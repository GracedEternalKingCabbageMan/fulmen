⚡ Fulmen SeqLN


      

        
Welcome

        
Fulmen is a desktop wallet for SeqLN, the Sequentia Lightning Network. It ships with a full SeqLN node inside: you don't install or configure anything. The node needs one thing from you, a Sequentia node it can read the chain through (yours, or any elementsd RPC you have access to).

        

          
          
        

      


      

        
Your Sequentia node

        
SeqLN reads the Sequentia chain through a node's RPC. Local or remote both work.
